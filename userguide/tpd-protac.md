# TPD / PROTAC

## 作用 / Role

TPD/PROTAC 模块用于后续 POI-E3 三元复合物、warhead、E3 ligand 和 linker 设计。

The TPD/PROTAC module is reserved for POI-E3 ternary complex, warhead, E3 ligand, and linker design workflows.

## 输入 / Inputs

- POI pocket asset
- E3 ligase structure
- POI warhead
- E3 ligand

## 输出 / Outputs

- PROTAC candidate assets
- ternary complex result assets

## API / Automation

当前为规划入口；后续任务会复用 `project_id`、`asset_id`、`job_id` 的统一自动化模型。
