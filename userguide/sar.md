# SAR / 构效关系

## 作用 / Role

SAR 模块整合活性数据、分子结构、对接/FEP 结果，用于药化决策和下一轮设计。

The SAR module combines activity data, molecular structures, docking/FEP results, and computed properties for medicinal chemistry decisions.

## 输入 / Inputs

- SMILES/SDF ligand assets
- activity table
- docking or FEP result assets

## 输出 / Outputs

- SAR table
- R-group/MMPA views
- next-design candidates

## API / Automation

当前为规划入口；配体资产和结果资产已经通过统一资产系统为 SAR 输入做准备。
